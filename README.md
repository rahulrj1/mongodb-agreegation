# MongoDB Aggregation Pipeline Examples

This repository contains examples of MongoDB aggregation pipelines.

## Files

- `users.json`: Sample user data.
- `books.json` & `authors.json`: Sample data for joins.
- `aggregation-examples.md`: A collection of aggregation pipeline examples with explanations.

## Usage

You can use these queries in:
- MongoDB Compass (Aggregation tab)
- MongoDB Shell (`mongosh`)

Just copy the array part of the query (the part inside `[]`) into your aggregation pipeline builder.
